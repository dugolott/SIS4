# SIS4
## Codici e progetti per il corso di Sistemi del 4° Anno

Repository contenente i codici di esempio e i progetti sviluppati durante il corso di Sistemi del 4° Anno dell'indirizzo Informatica e Telecomunicazioni dell'ITIS "L. Da Vinci (PARMA)".

I codici sono organizzati in cartelle corrispondenti agli argomenti trattati durante il corso, con sottocartelle per ogni lezione o progetto specifico.

### Struttura delle Cartelle
- `Programmazione C/`: Codici relativi agli argomenti di programmazione in linguaggio C.
- `Frame_Ethernet/`: progetto di studio del protocollo IEEE 802.3 (Ethernet).
- ...
